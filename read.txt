
1. Распределение ролей 
   Администратор 
   имеет абсолютные права: 
   - видит все заявки; 
   - дает одобрение на регистрацию нового пользователя;
   - удаляет (редактирует) пользователя;
   - удаляет (редактирует) заявки;
   - назначает исполнителя по заявке;
   - меняет исполнителя по заявке;
   - закрывает заявки, переводя их в категорию "Исполнено".

2. Пользователь
   имеет ограниченные права, установленные администратором:    
   - видит только заявки по которым он назначен исполнителем;
   - закрывает заявки, переводя их в категорию "Исполнено".

3. Регистрация пользователей
   Пользователь:
   - при регистрации указывает имя, телефон, email (опционно), придумывает пароль (первое поле) и подтверждает его (второе поле) /register;
   - система (backend) формирует ссылку с токеном, которая в элетронном письме с именем и номером телефона нового пользователя, направляется 
     Администратору для одобрения регистрации нового пользователя;
   - входит по имени и паролю /login.
